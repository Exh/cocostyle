csNumeral
=========

### 시작하기

```
        tl = new cc.LabelTTF(csNumeral(1000).format('0,0'), "Arial", 15);
        tl.x = winSize.width / 2;
        tl.y = winSize.height / 2 - 50;
        this.addChild(tl);
```
