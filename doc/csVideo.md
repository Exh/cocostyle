csVideo
=========

### Quick Start

```
var video = new csVideo('http://www.w3schools.com/html/mov_bbb.mp4');
this.addChild(video);
video.video.play();
```

### Fields

- {HTML5 Video} video
- And cc.Node fileds
- It's only work on HTML5.
- Required "frameworks/cocos2d-html5/extensions/editbox/CCdomNode.js"
